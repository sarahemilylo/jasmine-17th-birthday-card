An interactive birthday card website created by Sarah for Jasmine's 17th birthday.

This project is a single-page web application featuring animated gifts, personalized messages, photo memories, and interactive surprises designed to celebrate Jasmine's special day.

## ✨ Features

### 🎈 Welcome Screen
- Animated floating balloons
- Elegant pastel-themed design
- Personalized birthday greeting
- Interactive "Open Your Card" button

### 🔐 Secret Passcode Screen
- Custom keypad interface
- Birthday-based passcode challenge
- Animated passcode indicators
- Confetti celebration upon successful entry

### 🎁 Gift Hub
Unlock three unique birthday gifts:

#### 💌 Gift 1 — Letter & Birthday Cake
- Interactive envelope opening animation
- Personalized birthday letter from Sarah
- Animated birthday cake
- Clickable candles that can be blown out
- Confetti celebration after all candles are extinguished

#### 📸 Gift 2 — Memory Album
- Interactive Polaroid photo gallery
- Personalized captions
- Full-screen lightbox viewing
- Responsive photo layout
- Animated Polaroid-style cards

#### 🏺 Gift 3 — Appreciation Jar
- Animated message jar
- Randomized appreciation notes
- Jar shaking animation
- Multiple personalized messages
- Endless note drawing experience

## 🎉 Special Interactive Elements

### Confetti Engine
A custom canvas-based particle system is used to create celebration effects when:
- The correct passcode is entered
- All birthday candles are blown out

### Envelope Animation
The birthday letter is hidden inside a virtual envelope and animates upward when opened.

### Memory Album
Photos can be enlarged using a custom-built lightbox viewer.

### Appreciation Jar
Notes are randomly selected while preventing the same message from appearing twice in a row.
